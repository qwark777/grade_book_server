# Инструкция для школы: Как связать свою CRM с приложением

## Пошаговая инструкция

### Шаг 1: Войти в систему как администратор школы

1. Откройте приложение на телефоне/компьютере
2. Войдите в систему как администратор вашей школы
3. Перейдите в раздел **"Админ-панель"** или **"Настройки"**

### Шаг 2: Перейти в раздел "Интеграции CRM"

1. В админ-панели найдите раздел **"Интеграции CRM"** или **"Связь с CRM"**
2. Нажмите на него

### Шаг 3: Создать новую интеграцию

1. Нажмите кнопку **"Добавить интеграцию"** или **"Подключить CRM"**
2. Выберите тип вашей CRM системы:
   - **1С** - если у вас 1С:Бухгалтерия, 1С:Управление образовательной организацией
   - **Битрикс24** - если используете Битрикс24 CRM
   - **AmoCRM** - если используете AmoCRM
   - **Другая** - если у вас другая CRM система

### Шаг 4: Заполнить данные подключения

Вам понадобятся следующие данные из вашей CRM системы:

#### Для 1С:
- **Название интеграции**: например, "1С:Бухгалтерия школы №123"
- **URL API**: адрес веб-сервиса вашей 1С (например, `https://1c-school.example.com/api/v1`)
- **API ключ** или **Токен доступа**: токен для доступа к API вашей 1С

**Как получить токен в 1С:**
1. В 1С откройте раздел **"Администрирование"** → **"Интернет-поддержка"**
2. Создайте веб-сервис или используйте существующий
3. Скопируйте токен доступа или API ключ

#### Для Битрикс24:
- **Название интеграции**: например, "Битрикс24 CRM школы"
- **URL API**: URL вашего Битрикс24 (например, `https://school.bitrix24.ru/rest`)
- **Webhook ключ**: ключ вебхука из Битрикс24

**Как получить Webhook в Битрикс24:**
1. В Битрикс24 зайдите в **"Приложения"** → **"Разработчикам"** → **"Другое"** → **"Входящий вебхук"**
2. Выберите права доступа (нужны права на CRM, контакты, сделки)
3. Скопируйте URL вебхука

#### Для AmoCRM:
- **Название интеграции**: например, "AmoCRM школы"
- **URL API**: `https://ваш-домен.amocrm.ru/api/v4`
- **Access Token**: токен доступа из AmoCRM

**Как получить токен в AmoCRM:**
1. В AmoCRM зайдите в **"Настройки"** → **"Интеграции"** → **"API"**
2. Создайте интеграцию
3. Скопируйте Access Token

#### Для другой CRM (Custom):
- **Название интеграции**: название вашей CRM
- **URL API**: адрес API вашей CRM системы
- **API ключ/токен**: ключ доступа к API

### Шаг 5: Настроить параметры синхронизации

1. **Направление синхронизации**:
   - **"Из приложения в CRM"** - данные передаются из приложения в вашу CRM
   - **"Из CRM в приложение"** - данные передаются из CRM в приложение
   - **"Двусторонняя"** - синхронизация работает в обе стороны

2. **Частота синхронизации**:
   - **"В реальном времени"** - данные синхронизируются сразу при изменении
   - **"Каждый час"** - синхронизация раз в час
   - **"Ежедневно"** - синхронизация раз в день (рекомендуется)
   - **"Вручную"** - синхронизация только по нажатию кнопки

### Шаг 6: Настроить соответствие полей (Маппинг)

Если названия полей в вашей CRM отличаются от приложения, настройте соответствие:

**Пример для 1С:**
- `lesson.title` → `НазваниеЗанятия`
- `lesson.price` → `Цена`
- `student.full_name` → `ФИО_Студента`

**Пример для Битрикс24:**
- `lesson.title` → `TITLE`
- `lesson.price` → `OPPORTUNITY`
- `student.full_name` → `NAME`

### Шаг 7: Тестировать подключение

1. После заполнения всех полей нажмите кнопку **"Тестировать подключение"**
2. Дождитесь результата:
   - ✅ **"Подключение успешно"** - всё настроено правильно
   - ❌ **"Ошибка подключения"** - проверьте введенные данные

**Если ошибка подключения:**
- Проверьте правильность URL API
- Проверьте действительность API ключа/токена
- Убедитесь, что ваша CRM доступна из интернета
- Проверьте настройки безопасности в вашей CRM (белый список IP, разрешения API)

### Шаг 8: Активировать интеграцию

1. Если тест прошел успешно, включите переключатель **"Активировать интеграцию"**
2. Нажмите **"Сохранить"**

### Шаг 9: Запустить первую синхронизацию

1. После сохранения нажмите кнопку **"Синхронизировать сейчас"**
2. Выберите тип данных для синхронизации:
   - **Занятия** - дополнительные занятия
   - **Записи на занятия** - студенты, записавшиеся на занятия
   - **Финансовые данные** - платежи, доходы
   - **Полная синхронизация** - все данные сразу

### Шаг 10: Проверить результат

1. После синхронизации проверьте данные в вашей CRM
2. Убедитесь, что данные корректно импортированы
3. Проверьте историю синхронизаций в приложении

## Что синхронизируется

### Занятия (Lessons)
- Название занятия
- Предмет
- Преподаватель
- Цена
- Количество записанных студентов
- Дата создания и обновления

### Записи на занятия (Enrollments)
- ФИО студента
- Название занятия
- Дата записи
- Статус записи (записан, завершен, отменен)
- Статус оплаты (ожидает, оплачено, возврат)
- Сумма платежа

### Финансовые данные (Financial)
- Общий доход по занятиям
- Оплаченные платежи
- Ожидающие оплаты
- Просроченные платежи

## Частые вопросы

### Как получить API ключ/токен?

**Для 1С:**
1. Обратитесь к системному администратору 1С
2. Попросите создать веб-сервис для интеграции
3. Попросите токен доступа к API

**Для Битрикс24:**
1. Зайдите в "Приложения" → "Разработчикам"
2. Создайте входящий вебхук
3. Скопируйте URL вебхука

**Для AmoCRM:**
1. Зайдите в "Настройки" → "Интеграции" → "API"
2. Создайте интеграцию
3. Скопируйте Access Token

### Что делать, если CRM не подключается?

1. **Проверьте URL**: убедитесь, что адрес API правильный и доступен
2. **Проверьте токен**: убедитесь, что API ключ действителен и не истек
3. **Проверьте доступность**: убедитесь, что ваша CRM доступна из интернета
4. **Проверьте настройки безопасности**: в CRM должны быть разрешены запросы с нашего сервера
5. **Обратитесь в поддержку**: если проблема не решена, напишите в поддержку приложения

### Как часто синхронизируются данные?

Частота зависит от ваших настроек:
- **В реальном времени**: сразу при изменении данных
- **Каждый час**: автоматически каждый час
- **Ежедневно**: один раз в день (обычно ночью)
- **Вручную**: только когда вы нажимаете кнопку "Синхронизировать"

### Можно ли отключить интеграцию?

Да, вы можете в любой момент отключить интеграцию:
1. Зайдите в раздел "Интеграции CRM"
2. Найдите нужную интеграцию
3. Выключите переключатель "Активировать интеграцию"
4. Или удалите интеграцию полностью

### Безопасны ли мои данные?

Да, все данные передаются по защищенному соединению (HTTPS), а API ключи хранятся в зашифрованном виде. Доступ к настройкам интеграций имеют только администраторы школы.
