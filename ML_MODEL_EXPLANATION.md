# Объяснение работы ML модели для AI-советов

## 🧠 Как работает модель

### 1. Тип модели: RandomForestClassifier

Модель использует **Random Forest** (случайный лес) - это ансамбль из 50 простых деревьев решений, которые "голосуют" за итоговый приоритет.

### 2. На чем обучается модель (текущее состояние)

**⚠️ ВАЖНО**: Сейчас модель обучается на **синтетических (искусственных) данных**, а не на реальных данных студентов.

#### Процесс обучения:

1. **Генерация 1000 примеров** с разными характеристиками:
   - Средний балл: от 2.0 до 5.0
   - Процент пропусков: от 0% до 100%
   - Тренд: от -1.0 (сильное ухудшение) до +1.0 (сильное улучшение)
   - Количество проблемных предметов: от 0 до 5

2. **Разметка данных по правилам**:
   ```python
   # Высокий приоритет (2), если:
   - Средний балл < 3.0
   ИЛИ процент пропусков > 20%
   ИЛИ тренд < -0.3 (сильное ухудшение)
   ИЛИ проблемных предметов >= 3
   
   # Средний приоритет (1), если:
   - Средний балл < 3.5
   ИЛИ процент пропусков > 10%
   ИЛИ тренд < 0 (ухудшение)
   ИЛИ проблемных предметов >= 2
   
   # Низкий приоритет (0) во всех остальных случаях
   ```

3. **Обучение модели**: RandomForest изучает эти примеры и находит закономерности

### 3. Что анализирует модель для конкретного студента

Когда студент запрашивает советы, модель извлекает **4 признака**:

#### Признак 1: Средний балл
```
Средний балл = (сумма всех оценок) / (количество оценок)
```
Пример: если есть оценки [4, 5, 4, 3, 5], то средний = 4.2

#### Признак 2: Процент пропусков
```
Процент пропусков = (количество пропусков / общее количество уроков) * 100
```
Пример: 5 пропусков из 50 уроков = 10%

#### Признак 3: Тренд успеваемости
```
Тренд = последний средний балл - первый средний балл
```
Пример: было 3.5, стало 4.0 → тренд = +0.5 (улучшение)

#### Признак 4: Количество проблемных предметов
```
Проблемные предметы = количество предметов со средним баллом < 3.5
```
Пример: Математика (3.2), История (4.5), Физика (3.1) → 2 проблемных предмета

### 4. Предсказание модели

Модель получает 4 числа, нормализует их и предсказывает:
- **0** = Низкий приоритет (всё хорошо)
- **1** = Средний приоритет (есть проблемы, но не критичные)
- **2** = Высокий приоритет (серьезные проблемы!)

Также модель показывает **уверенность** (0-100%) в своем предсказании.

## 🎯 Почему модель выдала "Продолжайте в том же духе"

Судя по ответу, который вы видите в приложении, модель предсказала **низкий приоритет (0)**. Это означает, что для вашего студента:

### Вероятные характеристики:

✅ **Средний балл ≥ 3.5** (хорошая успеваемость)
✅ **Процент пропусков ≤ 10%** (хорошая посещаемость)
✅ **Тренд ≥ 0** (успеваемость стабильна или улучшается)
✅ **Проблемных предметов < 2** (мало или нет предметов с низкими оценками)

### Логика генерации совета:

Когда модель предсказала низкий приоритет И нет конкретных проблемных предметов, система генерирует общий позитивный совет:

```python
# Код из ai_advice.py (строки 185-196)

        id="advice_default",

        description="Ваша успеваемость стабильна. Продолжайте поддерживать текущий уровень.",
        priority="low",  # Низкий приоритет (от модели)
        category="study_habits",

        action_text=None
    ))
```
