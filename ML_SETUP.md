# Настройка ML модели для AI-советов

## ✅ Что было сделано

1. **Добавлены ML библиотеки**:
   - `scikit-learn==1.3.2` - для машинного обучения
   - `numpy==1.24.3` - для работы с массивами

## 📊 Как работает модель

### Признаки (features):
1. **Средний балл** - общий средний балл студента
2. **Процент пропусков** - процент пропущенных занятий
3. **Тренд** - изменение среднего балла (улучшение/ухудшение)
4. **Количество проблемных предметов** - предметы со средним баллом < 3.5

### Предсказание:
- **0 (низкий приоритет)** - все в порядке
- **1 (средний приоритет)** - есть проблемы, но не критичные
- **2 (высокий приоритет)** - серьезные проблемы, требуют внимания

### Уверенность модели:
Модель также показывает уверенность в предсказании (0-100%), которая отображается в описаниях советов.

## 🐛 Решение проблем

### Ошибка импорта sklearn/numpy/pandas:
```bash
pip install --upgrade scikit-learn numpy pandas
```

### Модель не сохраняется:
- Убедитесь, что у приложения есть права на запись в директорию `models/`
- Проверьте, что достаточно места на диске

### Модель работает медленно:
- Это нормально при первом запуске (обучение)
- После сохранения модель загружается быстро
- Можно уменьшить `n_estimators` в `ai_model.py` для ускорения

### Fallback на правила:
- Если модель не работает, система автоматически использует правило-ориентированный подход
- Это гарантирует, что советы всегда будут генерироваться
